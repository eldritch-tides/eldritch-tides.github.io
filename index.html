How to Play

Eldritch Tides runs locally — no server required beyond a one-line command.

Linux / Mac

cd eldritch-tides/
./serve.sh
# open http://localhost:8080
# saves → saves/captain.json

Windows

Double-click serve.bat
# open http://localhost:8080
Requires Python 3 (ships with most systems). The game uses ES modules which browsers block from file:// — the local server is the only requirement.

Wiki

Everything you need to survive the Pale Meridian's first voyage.

Sailing
Hold W to thrust, A/D to steer. Your ship has momentum — plan your turns early. Hull integrity degrades in storms and from cannon fire. Watch the HUD.
sailing · physics
🎣
Fishing Minigame
Press F to cast. Hold Space to raise the zone, release to let it fall. Keep the fish icon inside the green zone — tension fills when you do. Equip rods and bait from inventory for bonuses.
fishing · minigame
🌊
Weather & Panic
Night raises your panic meter. At 100% you die. Storms boost eldritch fish rates but accelerate panic. Dock before dark or carry emergency repair kits. The monster spawns when panic exceeds 40 at night.
survival · horror
⚓
Ports
Port Harrow — starting port, sells cargo, 3⊕/HP repairs, Chef Aldric & Harbormaster Voss for quests. Misthollow — cursed village east, 2⊕/HP repairs, witch sells eldritch bait and lore.
ports · quests
💀
The Monster
It spawns at night when panic > 40. Four phases: APPROACH → HUNT → LUNGE → RETREAT. Hull takes 18 damage on contact. Fire cannons to drive it back; it retreats after sustained hits. Despawns at dawn.
combat · boss
🔧
Hull & Repair
Warnings fire at 50%, 25%, and 10% hull. At 0% the run ends — save is deleted. Repair at port (costs gold per HP), or use an Emergency Repair Kit at sea (R) for 25HP. Kits sold by merchants.
survival · economy

Sailing
Gained passively while sailing. Higher levels reduce whirlpool danger and improve handling.

XP Curve: Eldritch Tides uses the RuneScape XP formula — each level requires roughly 10% more XP than the last. Level 99 requires approximately 13 million total XP. Eldritch catches give the most fishing XP; sinking the sea monster gives a large combat bonus.
Under the Hull
Source Code

Eldritch Tides is a vanilla JS game split across 29 ES modules. No build tools, no framework dependencies.

Running Locally

ES modules require HTTP — browsers block file:// imports. Run a local server:

# Linux / Mac
cd eldritch-tides/
./serve.sh

# Windows
serve.bat

# Then open http://localhost:8080
# Saves: saves/captain.json

Save Files

Saves are written to saves/captain.json — a plain JSON file you can back up, inspect, or share. The standalone HTML falls back to localStorage.

Building Standalone

Bundle everything — including enabled mods — into one HTML file:

python3 build.py
# → dist/eldritch-tides.html (no server needed)

Architecture

Each module has a single responsibility. Cross-module dependencies use explicit ES module imports.

Bundle strategy: build.py concatenates modules in dependency order, strips import/export keywords, and prepends a hoisted-globals block to avoid TDZ errors — producing a plain <script> with no module loader needed.

The Shipwright's Manual
Writing a Mod

Mods are plain JavaScript files that receive the EldrTidesAPI object. No build step, no bundler, no framework.

Create the Mod Folder

Create a folder inside mods/ with your mod's ID as the name:

mods/
  my-mod/
    mod.json
    mod.js

Write mod.json

{
  "id":          "my-mod",
  "name":        "My Mod",
  "version":     "1.0.0",
  "entry":       "mod.js",
  "description": "What your mod does."
}

Write mod.js

Export a default function that receives the API. Everything your mod does goes inside it:

export default function MyMod(api) {

  // Register a new fish species
  api.registerFish({
    name: 'Void Carp',
    icon: '🪸',
    type: 'eldritch',
    value: 400,
    spd:   3.5,
  });

  // Add an item to merchant stock
  api.registerMerchantStock(stock => {
    stock.push({
      id: 'void_lure', icon: '🌑', name: 'Void Lure',
      rarity: 'rare', type: 'bait',
      desc: 'Draws eldritch fish from the deep.',
      price: 180, qty: 2,
    });
    return stock;
  });

  // React to game events
  api.onEvent('catch', fish => {
    if (fish.type === 'eldritch') {
      api.log('The void stirs beneath the hull...', 'ev-eldritch');
    }
  });

  // Hook into the game loop
  api.onEvent('update', (dt) => {
    // dt = delta time in seconds
    // window.GAME has full game state
  });

  // Draw onto the canvas
  api.onEvent('draw', ({ ctx, W, H }) => {
    // ctx = CanvasRenderingContext2D
    // window._toScreen(wx, wy) → [sx, sy]
  });
}

Enable the Mod

Add your mod's ID to mods/index.json:

["my-mod"]

Multiple mods:

["pirate-life", "crabber-life", "my-mod"]
Dev server: refresh the browser — mods are loaded via fetch at runtime.
Standalone build: run python3 build.py to bake enabled mods into the HTML.

Disable a Mod

Remove it from mods/index.json:

[] // empty array = all mods disabled

Then rebuild or refresh. The mod folder stays in place — just not loaded.

Full Reference
EldrTidesAPI
api.registerFish(fishDef)
Add a new fish to the catch pool. Fields: name, icon, type ('fish'|'rare'|'eldritch'|'crab'), value (gold), spd (fishing difficulty 0.5–4.0).
api.registerItem(itemDef)
Add an item to ITEM_CATALOG. Appears in inventory and can be distributed via merchant stock hooks. Fields: id, name, icon, type, desc, effect.
api.registerQuest(questDef)
Add a quest to the quest board. Fields: id, name, npc, requirements (array of { fishName, qty }), reward (gold), desc.
api.registerMerchantStock(fn)
Hook into merchant stock generation. fn receives the current stock array and must return the (optionally modified) array. Called each time a merchant spawns.
api.registerIsland(islandDef)
Add a named island to the world. Fields: wx, wy, r (radius), type, label. Appears on the minimap once discovered.
api.log(msg, cls)
Write a line to the Captain's Log. CSS classes: 'ev-success' (green), 'ev-warn' (amber), 'ev-danger' (red), 'ev-system' (muted), 'ev-eldritch' (purple).
api.onEvent(event, callback)
Subscribe to a game event. Multiple mods can subscribe to the same event.
update(dt) draw({ctx,W,H}) keydown({code}) init catch(fish) dock death levelup daybreak nightfall
Return true from a keydown handler to consume the event (prevents the base game from also handling it).

Window Globals
Available at Runtime

These are exposed on window and accessible from any mod's update or draw handler:

window.GAME
Full game state object: x, y, hull, gold, cargo[], panic, isNight, day, and more.
window.MERCHANTS
Array of active merchant ship objects with position, state, and inventory.
window.CANNONBALLS
Player's active cannonball array. Check for collisions with custom entities.
window.PIRATE_BALLS
Active enemy cannonballs. Push to this array to fire at the player.
window._toScreen(wx, wy)
Convert world coordinates to screen pixel coordinates for drawing. Returns [sx, sy].
window._spawnSplash(wx, wy, col, r)
Spawn a water splash particle effect at world coordinates with given colour and radius.
window.SFX
Sound effects object: .cannon(), .dock(), .catchSound(), .warning(), .hit(), .splash().
window._addLog(msg, cls)
Direct log function (same as api.log but synchronous — use inside update handlers).
window.WORLD_OBJECTS
Array of world POIs: wrecks, kelp, buoys, stacks, caves. Each has wx, wy, type, r.
window.ISLANDS
Array of island definitions: wx, wy, r, type, label. Useful for proximity checks.
⚠ Limitation: Mods share the same global scope in the standalone build. Avoid naming variables the same as core game constants (GAME, WEATHER, PIRATES, etc.). Prefix your mod's state with a unique identifier.
The Mod Locker
Included Mods

Two mods ship with the game. Enable them individually or together in mods/index.json.

☠️
Pirate Life
v1.0.0 · pirate-life
Turn the tables. You're not defending against pirates — you are the pirate. Raid charter ships for gold, build a wanted level, and survive the Navy fleet sent to hunt you down.
  • Q near merchant opens Trade / Raid panel
  • Each raid raises wanted level 1–5 ★
  • Navy ships spawn faster and harder as stars climb
  • Sink a Navy ship: bounty gold + wanted level drops
  • B at port to bribe harbourmaster per star
  • Forged Papers in merchant stock at ★2+
  • Wanted 1–2 decays naturally after 90s clean sailing
// Enable in mods/index.json:
["pirate-life"]
🦀
The Simple Life of a Crabber
v1.0.0 · crabber-life
A passive, RuneScape-style fishing layer. Drop crab pots and sail away. Come back when they're full. No minigame — just patience, good spots, and the occasional crab incident.
  • G drops a pot at your position; G near a pot hauls it
  • Pots fill passively over 45–90s
  • Kelp beds (+60%), wrecks (+40%), buoys (+30%) boost fill rate
  • Deep water unlocks King Crab, Sailfish, Sea Turtle
  • 8% haul chance: crab incident deals hull damage
  • Pots despawn after 5 min if abandoned
  • Crabber's Kit upgrade (merchant) expands to 8 pots
// Enable in mods/index.json:
["crabber-life"]
Both mods can run simultaneously: ["pirate-life", "crabber-life"]. Drop pots while evading the Navy. The crabs don't care about your wanted level.

Soundtrack

Original music written for the voyage. The game layers a procedural ambient drone beneath the tracks — tension builds with panic, storm intensity darkens the harmony.

Tideglass Drift
2:13 · loops · plays while sailing
0:00 2:13
🔈
Written to sit beneath the procedural ambient layer — the synth drone shifts with storm intensity and panic while Tideglass Drift carries the melodic thread across calmer waters.
In-game: Music starts on first keypress. The procedural engine layers sine-wave drones, filtered noise, and plucked melody notes that respond to weather, panic level, and time of day. Tideglass Drift plays looped on top at 75% volume — drop your own MP3 into audio/ and update the filename in main.js.
