How to Play

Eldritch Tides runs locally — no server required beyond a one-line command.

Linux / Mac
cd eldritch-tides/
./serve.sh
# open http://localhost:8080
# saves → saves/captain.json
Windows
Double-click serve.bat
# open http://localhost:8080
Requires Python 3. The game uses ES modules which browsers block from file:// — the local server is the only requirement.

Source Code

Vanilla JS split across 29 ES modules. No build tools, no framework dependencies.

eldritch-tides / eldritch-tides-game ↗
Running Locally

ES modules require HTTP — browsers block file:// imports.

# Linux / Mac
cd eldritch-tides/
./serve.sh

# Windows
serve.bat

# open http://localhost:8080
# saves → saves/captain.json
Save Files

Saves write to saves/captain.json — plain JSON, easily backed up. Standalone HTML falls back to localStorage.

Building Standalone
python3 build.py
# → dist/eldritch-tides.html
Architecture

Each module has a single responsibility. Cross-module deps use explicit ES imports.

Bundle strategy: build.py concatenates modules in dependency order, strips import/export, prepends hoisted globals — plain <script>, no module loader.

The Shipwright's Manual
Writing a Mod

Mods are plain JS files receiving EldrTidesAPI. No build step, no bundler.

Create the Mod Folder

Create mods/my-mod/mod.json and mods/my-mod/mod.js

Write mod.json

{ "id":"my-mod", "name":"My Mod", "version":"1.0.0", "entry":"mod.js" }

Write mod.js

export default function MyMod(api) {
  api.registerFish({ name:'Void Carp', icon:'🪸', type:'eldritch', value:400, spd:3.5 });
  api.onEvent('catch', fish => { /* fish.type, fish.value */ });
  api.onEvent('update', dt => { /* window.GAME has full state */ });
}

Enable the Mod

Edit mods/index.json:

["my-mod"]
Dev: refresh browser. Standalone: run python3 build.py.

Window Globals
Available at Runtime

Accessible from any mod's update or draw handler:

window.GAME
Full game state: x, y, hull, gold, cargo[], panic, isNight, day…
window.MERCHANTS
Active merchant ship objects with position, state, inventory.
window.CANNONBALLS
Player's active cannonballs. Check for custom entity collisions.
window.PIRATE_BALLS
Enemy cannonballs. Push here to fire at the player.
window._toScreen(wx, wy)
World → screen pixels. Returns [sx, sy].
window._spawnSplash(wx, wy, col, r)
Spawn a water splash particle effect at world coordinates.
window.SFX
.cannon(), .dock(), .catchSound(), .warning(), .hit().
window._addLog(msg, cls)
Synchronous log — use inside update instead of api.log.
window.WORLD_OBJECTS
World POIs: wrecks, kelp, buoys. Each has wx, wy, type, r.
window.ISLANDS
Island defs: wx, wy, r, type, label. Useful for proximity checks.
⚠ Note: Mods share the same global scope in the standalone build. Prefix your mod's state with a unique identifier.
The Mod Locker
Included Mods

Two mods ship with the game. Enable them in mods/index.json.

☠️
Pirate Life
v1.0.0 · pirate-life
Turn the tables. You're not defending against pirates — you are the pirate.
  • Q near merchant opens Trade / Raid panel
  • Each raid raises wanted level 1–5 ★
  • Navy ships spawn faster per star
  • Sink a Navy ship: bounty gold + wanted drops
  • B at port to bribe harbourmaster
["pirate-life"]
🦀
The Simple Life of a Crabber
v1.0.0 · crabber-life
Drop crab pots and sail away. Come back when they're full. No minigame.
  • G drops a pot; G near pot hauls it
  • Pots fill passively over 45–90s
  • Kelp (+60%), wrecks (+40%), buoys (+30%)
  • Deep water unlocks King Crab, Sailfish, Sea Turtle
  • 8% crab incident deals hull damage
["crabber-life"]
Both run together: ["pirate-life", "crabber-life"]. The crabs don't care about your wanted level.
